# CarIQ: Global & India Car Sales + Affordability Intelligence

## System Architecture
Dual-codebase integrated capstone project.

- `/app` — Express.js operational web application
- `/analytics` — Python data pipeline, EDA, and dashboards

## Setup Instructions
### App (Node.js)
```bash
cd app && npm install && npm run dev
```

### Analytics (Python)
```bash
cd analytics && source .venv/Scripts/activate && jupyter notebook
```
